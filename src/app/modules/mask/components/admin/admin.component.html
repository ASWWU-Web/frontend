Batch Upload Mask Photos

  • Clean up your excel file by deleting all columns and information except for student id, and the photo numbers.
  • Column one should contain student id's, and column two should contain the photo number(s) associated with the student id.
  • Make sure that the photo number(s) are separated with a space instead of a comma (can use replace all).
  • Save your excel file as a comma delimited csv file (in excel click `save as`, change the file format to Comma Separated Values (.csv), then save your new file).
  • Upload CSV file below, and fix any errors that pop up.
  • Select Photos that you would like to upload (You can only upload 50 photos at a time, but you can use the same csv file for each upload)
  • Do not close the page until all photos have finished uploading.
  • Refer to the photo for an example csv file.
